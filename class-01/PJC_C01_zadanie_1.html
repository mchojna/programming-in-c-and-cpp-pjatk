import java.util.Scanner;


public class Main {

    public static void main(String[] args) {

        Scanner input = new Scanner(System.in);

        int numberToGuess = (int)(Math.random() * 100);

        int guessedNumber = -1;


        while (guessedNumber != numberToGuess) {

            System.out.print("Guess a number: ");


            guessedNumber = input.nextInt();


            if (guessedNumber > numberToGuess) {

                System.out.println("Too big. Try lower.");

            }

            if (guessedNumber < numberToGuess) {

                System.out.println("Too small. Try higher.");

import java.util.ArrayList;

import java.util.List;

import java.util.Scanner;


public class Main {

    public static void main(String[] args) {

        Scanner input = new Scanner(System.in);

        System.out.println("Please input the numbers.");

        System.out.println("Type in a negative number to stop.");


        List<Integer> numbersToCheckForPrimality = new ArrayList<>();


        for (

                int number = input.nextInt();

                number >= 0;

                number = input.nextInt()

        ) {

            numbersToCheckForPrimality.add(number);

        }


        for (int number : numbersToCheckForPrimality) {

            if (isPrime(number)) {

                System.out.println(number + " is prime.");

            } else {

                System.out.println(number + " is not prime.");

            }

        }

    }


    public static boolean isPrime(int number) {

        if (number < 2) return false;

        for (int i = 2; i <= Math.sqrt(number); i++) {

            if (number % i == 0) return false;

        }

        return true;

    }

}
Zwróć uwagę na różnice:
  • w C++ nie ma List<> ani ArrayList<>. W celu stworzenia kolekcji, która przechowuje zmienną liczbę elementów (czyli wiemy, że zwykła tablica się nie nada), użyj std::vector.
  • W C++ funkcje pomocnicze nie muszą być metodami klas (a więc nie trzeba używać modyfikatora static).
  • Jak dodasz nowy element do wektora?
  • Gdzie stworzysz funkcję pomocniczą?


  1. std::rand() jest bardzo niskiej jakości generatorem liczb pseudolosowych. W przypadku bardzo prostych programów można go używać, lecz w przypadku wyszukanych symulacji, na podstawie których chcemy wysnuć jakieś wnioski dotyczące badań naukowych, zdecydowanie lepiej jest użyć wysokiej jakości elementów z biblioteki <random>. Niestety biblioteka ta jest bardzo trudna w użyciu dla początkujących. ↩
