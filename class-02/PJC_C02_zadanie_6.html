Sortowanie wybranej połowy wektora

Niekiedy chcemy zaaplikować jakiś algorytm tylko do części danych, a nie do nich wszystkich. Przykładowo, chcemy posortować tylko połowę kontenera. Tę połowę, której suma elementów będzie większa od drugiej.

Napisz funkcję sortBiggerHalf(), która przyjmie przez argument wektor liczb. Funkcja powinna sprawdzić, która jego połowa zawiera elementy o większej sumie. Posortuj tę połowę.

Jeżeli wektor zawiera nieparzystą liczbę elementów, to któraś z “połów” będzie większa o 1 element. Nie jest istotne, która.
Jeżeli elementy w obydwu połowach sumują się do takich samych wartości, to posortuj dowolną z połówek.
Nie twórz osobnego wektora. Modyfikuj przekazany oryginał.

auto main() -> int {

    auto firstHalfBigger  = std::vector<int>{3, 2, 1, 0, 1, 0};

    auto secondHalfBigger = std::vector<int>{3, 2, 1, 6, 5, 4};

    auto bothHalvesSame   = std::vector<int>{5, 4, 5, 4};


    sortBiggerHalf(firstHalfBigger);

    sortBiggerHalf(secondHalfBigger);

    sortBiggerHalf(bothHalvesSame);


    fmt::println(

            "{}\n{}\n{}",

            firstHalfBigger,

            secondHalfBigger,

            bothHalvesSame

    );

}
