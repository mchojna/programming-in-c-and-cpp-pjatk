Sprawdzanie, czy dwie linie są równoległe

Rozważmy następujące funkcje:

\[\begin{equation} f(x) = \frac{1}{2} x - \frac{1}{2} \label{eq:fun1} \end{equation}\]

oraz

\[\begin{equation} f(x) = \frac{1}{2} x + 1\frac{1}{2} \label{eq:fun2} \end{equation}\]

Linie powstałe poprzez naszkicowanie wykresów funkcji \eqref{eq:fun1} i \eqref{eq:fun2} są równoległe. Pierwsza z nich przecina, między innymi, punkty \((3, 1)\) oraz \((5, 2)\), a druga \((1, 2)\) oraz \((7, 5)\).

Dostarcz operator ||, który, wywołany dla dwóch obiektów typu Line, sprawdzi, czy są one równoległe. Przykładową strukturę Line oraz przykładowe wywołanie kodu podano poniżej:

#include <fmt/format.h>

struct Point {
    int x;
    int y;
};

struct Line {
    Point first;
    Point second;
};

// missing operator||()

auto main() -> int {
    auto l1 = Line{ {3, 1}, {5, 2} }; // aforementioned points
    auto l2 = Line{ {1, 2}, {7, 5} }; // aforementioned points

    fmt::println("{}", l1 || l2);

    auto l3 = Line{ {3, 1}, {5, 1} };
    auto l4 = Line{ {1, 7}, {-9, 7} };

    fmt::println("{}", l3 || l4);

    fmt::println("{}", l1 || l4);
}
